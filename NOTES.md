# NOTES — 출처와 범위

## 이게 뭔가
한 연구자의 Claude Code 운영 설정(전역 지침·스킬·에이전트·누적 메모리)에서 **재사용 가능한 방법론만 추출·익명화**한 포터블 세트입니다. 특정 대화에 첨부하거나 Claude Code에 설치해 씁니다.

## 익명화·제외 범위 (PUBLIC)
아래는 이 배포본에서 **의도적으로 제거/일반화**되었습니다:
- 개인 식별정보(실명·소속·이메일) — 미포함.
- 개인 파일 경로·동기화 폴더(메모리 정본 위치 등) — 일반 placeholder로 대체.
- 특정 기관의 접근/자격 세부(예: 기관 프록시·IP 접근) — 트리거와 하드-원 교훈만 남기고 세부 제거.
- 내부 렌더/보안 도구·바이너리 경로 — 미포함. 이름 언급도 최소화.
- 특정 도메인(연구 분야·프로젝트명) 규칙 — 코어에서 제외하고 "도메인 레이어는 직접" 안내로 대체.
- 토큰·API키·비밀번호 — 원본에도 없었고, 스캔 결과 없음.

## 신뢰 근거
규칙 대부분은 취향이 아니라 **실제로 파일이 깨지거나·오류가 새거나·독자가 헷갈렸던 시행착오**에서 나왔습니다(코어 §0 참조). 검증 습관(렌더·추출·대조)은 문서로 대체되지 않으니 코어 §4를 특히 지키세요.

## 기여·수정
규칙은 기본값이며 각자의 작업 맥락에 맞게 덮어써서 쓰는 것을 전제로 합니다. 개선 제안 환영.
